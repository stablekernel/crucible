Crucible ships runnable examples that double as the suite's living documentation.
The flagship is the **food-delivery order** machine in `examples/fooddelivery`.

## Order lifecycle

The diagram below is generated at build time from the real `fooddelivery.NewModel()`
machine — the same machine the example's tests exercise — so it can never drift from
the code. It exercises a hierarchical fulfillment flow, a parallel watchdog region,
a registry-bound guard, and timer-driven escalation.

visualization API.
